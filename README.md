# Credit Card Default Prediction

Machine learning project predicting credit card default risk using classification models.

## Tech Stack
Python, Scikit-learn, XGBoost, Pandas, Matplotlib

## Models Used
- Logistic Regression
- Decision Tree
- XGBoost

## Results
Best model achieved ROC-AUC score of 0.79.

## How to run

1. Clone repo
`git clone https://github.com/Soheb-Khann/Default-Credit-Card-Prediction.git`

2. Install dependencies
`pip install -r requirements.txt`

3. Run notebook
`jupyter notebook`
